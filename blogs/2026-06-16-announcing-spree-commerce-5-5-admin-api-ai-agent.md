Spree Commerce 5.5 makes building on the platform faster for both humans and AI agents. A new Admin API with a fully-typed TypeScript SDK opens every back office operation to integrations, and the release adds AI Agent Skills and Sales Channels so multi-channel sellers can manage everything from one backend with improved order routing.
